## 1. 수집하는 개인정보 항목

### 1.1 필수 수집 항목

#### (1) 계정 정보
- 이메일 주소
- 비밀번호 (암호화 저장)
- 사용자 고유 식별자 (user_id)

#### (2) 소셜 로그인 정보 (OAuth 연동 시)
- **Google OAuth**:
  - Google 계정 식별자 (sub)
  - 이메일 주소
  - 프로필 정보 (이름, 프로필 이미지 URL)
  - OAuth 액세스 토큰 및 리프레시 토큰
  - 토큰 만료 시각

#### (3) YouTube 연동 데이터
본 서비스는 YouTube Data API v3를 사용하여 다음 정보를 수집합니다:

- **채널 정보**:
  - YouTube 채널 ID 및 채널명
  - 구독자 수, 영상 수, 조회수 등 통계 정보
  - 채널 설명, 프로필 이미지 URL
  
- **영상 정보**:
  - 영상 ID, 제목, 설명
  - 게시 일시, 조회수, 좋아요/싫어요 수
  - 썸네일 URL
  - 영상 자막/스크립트 (yt-dlp 사용)

- **댓글 정보**:
  - 댓글 ID, 댓글 내용
  - 댓글 작성자 채널명 및 ID
  - 댓글 작성 시각
  - 좋아요 수, 답글 수
  - 댓글 상태 (공개/비공개)

#### (4) 필터링 관련 데이터
- **사용자 설정**:
  - 필터링 카테고리 선택 (욕설, 혐오 표현, 인신공격, 외모 비하, 성적 표현, 스팸, 일반)
  - 커스텀 키워드 목록
  - 예시 댓글 선택 (차단/허용)
  
- **필터링 결과**:
  - AI 모델의 댓글 분류 결과
  - 필터링 정책 (JSON 형식)
  - 일일 댓글 통계 (총 댓글 수, 필터링된 댓글 수)

#### (5) 서비스 이용 로그
- 접속 일시 및 IP 주소
- 브라우저 종류 및 OS 정보
- 이용 기록 (API 호출 로그, 동기화 기록)
- 오류 로그 및 디버그 정보

### 1.2 선택 수집 항목
- 닉네임
- 프로필 이미지
- 설문조사 응답
- 추가 피드백 및 제안사항

## 3. 개인정보의 보유 및 파기 기간

### 3.1 베타 테스트 운영을 위한 기본 원칙
- **베타 테스트 종료 후**, 수집된 개인정보는 테스트 종료일로부터 **30일 이내에 모두 파기**합니다.
- 단, 사용자가 정식 서비스 전환에 동의한 경우, 해당 정보는 정식 서비스로 승계될 수 있습니다.

### 3.2 항목별 보유 기간

| 항목 | 보유 기간 | 비고 |
|------|----------|------|
| 계정 정보 | 회원 탈퇴 시 즉시 파기 | OAuth 토큰은 즉시 무효화 |
| YouTube 댓글 데이터 | 베타 종료 후 30일 | Redis 캐시는 30일 TTL |
| 필터링 설정 | 베타 종료 후 30일 | |
| 서비스 이용 로그 | 베타 종료 후 30일 | |
| AI 학습 데이터 | 익명화 후 보관 가능 | 개인 식별 정보 제거 |

### 3.3 예외적으로 보존이 필요한 경우
관계 법령에서 별도의 보존 기간을 정하고 있는 경우, 해당 법령에서 정한 기간 동안 보관할 수 있습니다:

- **전자상거래 등에서의 소비자 보호에 관한 법률**:
  - 계약 또는 청약철회 등에 관한 기록: 5년
  - 대금결제 및 재화 등의 공급에 관한 기록: 5년
  - 소비자의 불만 또는 분쟁처리에 관한 기록: 3년

- **통신비밀보호법**:
  - 서비스 이용 관련 로그 기록: 3개월

### 3.4 파기 방법
- **전자적 파일**: 복구 불가능한 기술적 방법으로 완전 삭제
- **출력물 등**: 분쇄 또는 소각을 통해 파기
- **데이터베이스**: 물리적 삭제 및 백업 파일 제거
- **Redis 캐시**: TTL 만료 또는 수동 삭제

- 법령에 근거가 있거나 수사기관·법원 등의 정당한 요청이 있는 경우

**회사는 위 경우를 제외하고는 이용자의 동의 없이 제3자에게 개인정보를 제공하지 않습니다.**

## 7. 쿠키(Cookie) 등 사용에 관한 사항

### 7.1 쿠키 사용 목적
회사는 서비스 이용 편의 및 통계 분석을 위해 쿠키를 사용할 수 있습니다:

- 로그인 상태 유지
- 사용자 설정 저장
- 이용 패턴 분석 및 서비스 개선
- 세션 관리

### 7.2 쿠키 설치·운영 및 거부
- **쿠키 확인**: 웹 브라우저 설정에서 쿠키 저장 내역을 확인할 수 있습니다.
- **쿠키 거부**: 브라우저 설정에서 쿠키 저장을 거부하거나 삭제할 수 있습니다.
  - Chrome: 설정 > 개인정보 및 보안 > 쿠키 및 기타 사이트 데이터
  - Firefox: 설정 > 개인정보 및 보안 > 쿠키 및 사이트 데이터
  - Safari: 환경설정 > 개인정보 > 쿠키 및 웹사이트 데이터

- **거부 시 영향**: 쿠키 거부 시 로그인 유지, 일부 맞춤 서비스 이용에 제한이 있을 수 있습니다.
